Este código CSS va a crear un portafolio simple y minimalista..

... con un header, una sección para proyectos, y un footer.
Los proyectos se muestran como tarjetas con una imagen, un titulo, y una descripción.
La fuente es Arial, y el fondo es un gris claro.
El header y el footer tienen un fondo gris oscuro con texto blanco.
Los proyectos son mostrados con un fondo blanco y una leve sombra para agregar profundidad.
Feel free to modify the code to fit your needs and personal style.
